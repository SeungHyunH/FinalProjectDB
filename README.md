	  - 우선사항3 : 제일 좋아하는 게임인만큼 관심도가 매우 높음

	3. LOL : 챔피언 - 아이템 - 룬
	  - 우선사항1 : Join으로 출력할 요소가 애매함
	  - 우선사항2 : 확장시키기는 좋음. 인기가 제일 많은 게임인 만큼 여러 주제들이 있고, 이후 라이엇API를 사용해서 확장시킬 수 있음
	  - 우선사항3 : 아무래도 자주하는 게임 중 하나라서 관심도도 높음
